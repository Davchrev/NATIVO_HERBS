# Imágenes de destinos representativos

Coloca aquí un PNG **sin fondo (transparente)** por cada región, con el destino
más representativo (Machu Picchu, Cabeza Clava, Tumi de Oro, etc.).

## Reglas importantes
- El nombre del archivo debe coincidir **EXACTAMENTE** (minúsculas, con guiones).
  GitHub distingue mayúsculas/minúsculas: `cusco.png` ≠ `Cusco.PNG`.
- Formato: `.png` con transparencia.
- Si falta una imagen, la web simplemente no la muestra (no da error).
